# 📚 Estructuras de Datos - UNED

Repositorio dedicado a la implementación y testeo de estructuras de datos, siguiendo la metodología de la lista de reproducción de **Makigas**.

## 🎯 Objetivo
Desarrollar implementaciones robustas y eficientes de las estructuras clásicas, asegurando su correcto funcionamiento mediante **pruebas unitarias**.

## 🛠️ Estructuras a Implementar
* **Lineales:** Listas enlazadas (simples/dobles), Pilas (Stacks) y Colas (Queues).
* **No Lineales:** Árboles binarios, Árboles de búsqueda (BST) y Grafos.
* **Algoritmos:** Ordenación, búsqueda y recorridos.


## 🧪 Plan de Pruebas (Tests)
Para cada estructura se validarán los escenarios críticos de cada uno de los puntos importantes de la playlist.∫
